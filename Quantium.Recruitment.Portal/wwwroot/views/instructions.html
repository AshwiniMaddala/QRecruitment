Instructions
  • You can view the number of questions and total time for test one left side of the page

  • You have to answer the question before the timer runs out
  • Do not click 'FINISH TEST' unless you want complete the test

  • If you LOGOUT, the current question will be marked as Answered along with the options you selected
  • If you LOGOUT, the current question will not be visible the next time you login
  • Questions can be from various domains
  • You cannot view the previous questions once you go to the next question
  • There could be multiple answers for each question
